# 🧠 常識・雑学クイズアプリ（Streamlit）

このアプリは、30問の常識・雑学クイズに挑戦できる Web アプリです。  
Streamlit を用いて作成しており、回答の正誤判定やスコア集計、ヒント機能を備えています。  
また、回答履歴は Supabase に保存されます。

---

## 📌 アプリの主な機能

- 4択形式の雑学クイズ（全30問）
- 問題番号の表示（第1問〜第30問）
- 各問題にヒント機能を搭載
- 回答ごとの正誤判定を即時表示
- 正解数・正答率・使用ヒント数の集計
- Supabase による回答ログの保存
- 全問終了後に結果画面を表示

---

## 🎮 使い方

1. ユーザー名を入力（未入力でもプレイ可能）
2. 問題文を読み、選択肢から回答を選ぶ
3. 必要に応じてヒントを見る
4. 「回答する」を押して正誤を確認
5. 「次の問題へ」を押して次に進む
6. 30問すべて解き終えると結果が表示される
